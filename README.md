# Aplikasi untuk live tracking kecepatan Menulis
Kadang kita nulis suka ga konsen, ngelantur, nah trakcing ini bisa bikin lebih fokus lagi dalam menulis!
## Fitur Fitrunya
1. Live bisa tahu berapa kecepatan mengetik kita WPM/KPM
2. Bisa tau berapa jumlah kata
3. Bisa ada tracking waktunya
4. Ada panel penyimpan riwayat teks yang ditulis setiap sesi
